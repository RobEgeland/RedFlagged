# Features Layout: Free vs Paid

## FREE TIER FEATURES

### Core Analysis
- ✅ **Deal/Caution/Disaster Verdict**
  - Overall recommendation with confidence score
  - Basic explanation of verdict reasoning
  - Price context (if price is significantly above/below market)

- ✅ **Price vs Market Comparison**
  - Asking price display
  - Estimated market value
  - Price difference (dollar amount and percentage)
  - Basic pricing comparison using Edmunds/KBB data

- ✅ **Red Flags Identification**
  - All red flags detected (unlimited count)
  - Flag title and basic description
  - Severity level (critical/high/medium/low)
  - Category classification
  - **Note:** Expanded details and methodology are hidden in free tier

- ✅ **Basic Questions to Ask Seller**
  - 1-2 essential questions
  - Always includes title verification question
  - One additional question based on highest priority red flag

- ✅ **Vehicle Information Display**
  - Year, Make, Model, Trim
  - Mileage
  - VIN (if provided)
  - Asking price

- ✅ **Data Quality Assessment**
  - Overall confidence level (high/medium/low)
  - Confidence score percentage
  - Data quality factors
  - Recommendations for improving data quality

- ✅ **Transparency Disclosure**
  - What data sources were checked
  - What couldn't be checked
  - Known data vs unknown data
  - **Note:** Limited data sources shown (NMVTIS, basic pricing, FEMA disaster data)

- ✅ **Environmental Risk Assessment**
  - Overall risk level (low/medium/high)
  - FEMA disaster declarations (last 5 years)
  - Disaster history summary
  - Risk factors identified

- ✅ **Vehicle Recalls**
  - Open recalls display
  - NHTSA campaign numbers
  - Component affected
  - Summary, consequence, and remedy details
  - Report received dates

- ✅ **Upgrade Prompt**
  - Call-to-action to upgrade to premium
  - Feature comparison list

---

## PAID TIER FEATURES

### All Free Features Included PLUS:

### Enhanced Verdict & Analysis
- ✅ **Enhanced Verdict Explanation**
  - Detailed context from premium features
  - Market pricing analysis integration
  - Maintenance risk context
  - Environmental risk context
  - Data quality impact notes

### Premium Data Sources
- ✅ **Complete Vehicle History Analysis**
  - Full Carfax/AutoCheck data (not just NMVTIS)
  - Comprehensive accident history
  - Ownership pattern analysis
  - Service record indicators
  - Title history details
  - Vehicle History Summary card

- ✅ **Enhanced Market Data**
  - Auto.dev active listings
  - MarketCheck sales statistics
  - Mileage-weighted market averages
  - Raw listing data for detailed analysis

- ✅ **Expanded Disaster Data**
  - FEMA disaster declarations
  - NOAA storm data
  - USGS wildfire data
  - Comprehensive environmental risk assessment

### Premium Analysis Features

- ✅ **Market Pricing Analysis Card**
  - Fair price ranges (low, median, high)
  - Percentile positioning (where asking price falls)
  - Comparable listings count
  - Geographic scope
  - Listing type breakdown (private-party vs dealer)
  - Negotiation leverage assessment (strong/neutral/weak)
  - Market summary and insights

- ✅ **Maintenance Risk Assessment Card**
  - Overall risk classification (low/medium/elevated)
  - Risk classification explanation
  - Risk factors by component
  - Inspection focus areas (high/medium/low priority)
  - Buyer checklist
  - Confidence level and limitations
  - Forward-looking maintenance concerns

- ✅ **Tailored Questions Card**
  - Comprehensive, report-specific questions
  - Questions organized by priority:
    - Critical priority
    - High priority
    - Medium priority
    - Low priority
  - Questions categorized by type:
    - Title & History
    - Pricing
    - Maintenance
    - Condition
    - Seller
    - Environmental
    - General
  - Context for each question (why it matters)
  - Related findings that triggered each question
  - Summary of question strategy

- ✅ **Seller Signals Analysis**
  - Listing behavior analysis
    - Relisting detection
    - Listing longevity (stale listing detection)
    - Price volatility tracking
  - Pricing behavior analysis
    - "Too good for too long" detection
    - Price change patterns
  - Profile consistency analysis
    - Dealer curbstoning detection
    - Profile verification
  - Seller credibility score (0-100)
  - Seller insights and recommendations

- ✅ **Seller Credibility Score Card**
  - Numerical credibility score (0-100%)
  - Visual progress bar
  - Detailed insights explaining the score
  - Factors affecting credibility

- ✅ **Comparable Listings**
  - 3-5 local comparable listings
  - Price, mileage, location
  - Days on market
  - Source attribution

- ✅ **Enhanced Red Flags**
  - All red flags with expanded details
  - Full methodology explanations
  - Data source attribution
  - Expanded context and recommendations

- ✅ **Enhanced Transparency Disclosure**
  - Complete data source list:
    - Vehicle history: Carfax/AutoCheck/NMVTIS databases
    - Market data: Auto.dev, MarketCheck, Edmunds, KBB
    - Disaster data: FEMA, NOAA, USGS
    - Seller signals: Listing behavior analysis
  - Full known/unknown data breakdown

### Premium Verdict Logic
- ✅ **Advanced Verdict Assembly**
  - Considers maintenance risk assessment
  - Incorporates market pricing analysis
  - Factors in environmental risk levels
  - Premium features can influence verdict:
    - Elevated maintenance risk can prevent "Deal" verdict
    - Unfavorable market position can trigger "Caution"
    - Premium signals can contribute to "Disaster" verdict

## FEATURE COMPARISON SUMMARY

| Feature | Free Tier | Paid Tier |
|---------|-----------|-----------|
| **Verdict** | ✅ Basic | ✅ Enhanced with premium context |
| **Price Comparison** | ✅ Basic (Edmunds/KBB) | ✅ Detailed (Auto.dev, MarketCheck, weighted averages) |
| **Red Flags** | ✅ All flags, basic details | ✅ All flags, expanded details + methodology |
| **Questions** | ✅ 1-2 essential | ✅ Comprehensive tailored questions (8-15+) |
| **Vehicle History** | ✅ NMVTIS only | ✅ Carfax/AutoCheck/NMVTIS full analysis |
| **Market Pricing Analysis** | ❌ | ✅ Detailed percentile, ranges, leverage |
| **Maintenance Risk** | ❌ | ✅ Forward-looking assessment |
| **Seller Analysis** | ❌ | ✅ Credibility score + behavior signals |
| **Comparable Listings** | ❌ | ✅ 3-5 local comparables |
| **Environmental Risk** | ✅ Basic (FEMA) | ✅ Comprehensive (FEMA + NOAA + USGS) |
| **Data Quality** | ✅ Basic | ✅ Enhanced with recommendations |
| **Transparency** | ✅ Limited sources | ✅ Complete source breakdown |

---

## NOTES

- **Free tier** provides essential information to make a basic decision
- **Paid tier** provides comprehensive analysis with forward-looking risk assessments and detailed market intelligence
- All premium features are integrated into the verdict logic for more accurate recommendations
- Premium features never independently change verdict but provide important context and can influence the final decision
